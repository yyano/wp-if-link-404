if link is 404(broken), then change post status.

== Description ==
If the link destination is 404(broken), 503(Service Temporarily Unavailable) and any errors, then update the post status.

When the link destination is no longer displayed, you need to update your own page.

With this plugin, articles with broken links are automatically set to non-public post status(Draft/Private/Pending).
so that you can keep healthy content at any time.

== Installation ==
1. Use WordPress Plugins, "Add New" feature, searching "if link is 404".
1. Activate the plugin through the 'Plugins' menu in WordPress.
